# Mechanic Barabarani – Garaji Mahali Popote

A JavaScript capstone project that helps Kenyan drivers find trusted mechanics quickly.

- HTML5 + Tailwind CSS (via CDN)
- Vanilla JavaScript (DOM, fetch API, local storage)
- JSON “database” with 30 mechanics from Nairobi, Mombasa, Kisumu, Nakuru, Machakos
- Features:
  - Search and filter by city and service
  - Emergency button (closest available mechanics)
  - Favourites via localStorage
  - Register Mechanic form (Sajili Mekani – mock‑admin)
